# Third-Party Open Source Projects

JPetRemAI uses the following open source projects. We're grateful to all the maintainers and contributors.

---

## Core Dependencies

### [Shimeji-ee](https://github.com/hyakumeguri/shimeji-ee)
- **License**: BSD License
- **Description**: Desktop mascot engine for Windows/Mac/Linux
- **Usage**: Core desktop pet rendering and behavior system

### [llama.cpp](https://github.com/ggerganov/llama.cpp)
- **License**: MIT License
- **Description**: Port of GGML/GGUF LLM inference to pure C/C++
- **Usage**: Local LLM inference engine (llama-server)

### [Javassist](https://www.jboss.org/javassist)
- **License**: MPL 1.1 / LGPL 2.1
- **Description**: bytecode manipulation toolkit for Java
- **Usage**: Hot-patching JAR files without recompilation

### [FlatLaf](https://www.formdev.com/flatlaf/)
- **License**: Apache License 2.0
- **Description**: Flat, modern Look and Feel for Java Swing
- **Usage**: UI theming and styling

### [CFR](https://github.com/leibnitz27/cfr)
- **License**: MIT License
- **Description**: Java decompiler
- **Usage**: Development tool for bytecode analysis

---

## UI/UX Components

### [Apple Swing Extensions](https://developer.apple.com/documentation)
- **License**: Apple Developer SDK
- **Description**: Native macOS UI integration for Java
- **Usage**: Window decorations, blur effects, system integration
